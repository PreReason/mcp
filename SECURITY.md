# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email us at: **security@prereason.com**

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This package (`@prereason/mcp`) is a lightweight stdio-to-HTTP bridge. It forwards MCP messages between a local MCP client (e.g. Claude Desktop) and the PreReason API at `https://api.prereason.com/api/mcp`.

Security concerns related to the PreReason API itself should also be reported to security@prereason.com.
